# workpg

Child-friendly 2D top-down art gallery game.

**Now features:**
- 100 connected rooms with beautiful carpet flooring
- 30 AI characters walking around the gallery
- 3 interactive characters to talk to (Alice, Bob, and Charlie)
- Explore and discover all the artwork across the entire gallery!

## Run

Use any static file server from this folder, for example:

```bash
npx serve .
```

Then open the shown local URL in your browser.

## Features

- **100 Rooms**: Massive gallery to explore with varied artwork
- **AI Characters**: 30 NPCs walking throughout the gallery
- **Carpet Flooring**: Beautiful carpet pattern on all floors
- **Interactive NPCs**: Talk to 3 special characters for hints and messages
- **Smooth Movement**: Fluid controls for navigating the gallery

## Controls

- `Arrow Up/Down`: move forward/backward
- `Arrow Left/Right`: turn left/right
- **Walk near interactive characters** to see their messages
